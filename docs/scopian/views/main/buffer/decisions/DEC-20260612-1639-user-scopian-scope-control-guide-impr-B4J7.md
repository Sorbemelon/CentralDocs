# Decision: Scopian scope control guide improvement

Scopian was previously bypassed because Codex created private Markdown planning files directly instead of using Scopian Buffer. Guide improvement needed: new/additional/deviating context should become Buffer first; guard evidence is not edit permission; new source files and source registry changes require explicit user approval; source sync should follow Scopian Buffer -> user decision -> source edit -> view refresh.
